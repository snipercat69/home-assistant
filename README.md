# home-assistant

Home Assistant smart home integration scripts for Discord. Control lights, switches, thermostats, vacuum robots, and other smart devices via natural-language commands in Discord.

## Topics
python, home-assistant, smart-home, iot, automation, discord

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/home-assistant.git
cd home-assistant

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 home-assistant.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
